Feature map

Every Luvi area has one job and a fast path.

Luvi avoids bloated enterprise screens. Each section is tuned for repeated desktop use: scan, decide, act, move on.

Boards

Kanban, list, and table views

Create tasks, move work, filter by priority, assign teammates, and open detailed task modals without leaving the project.

Team Chat

Project rooms plus direct messages

Every project creates a private chat room. Direct messages keep person-to-person coordination separate from project work.

Inbox

Requests and due work

Team invites, transfer signals, and upcoming tasks live in a focused inbox so managers do not hunt through chat.

Goals

Progress without clutter

Goal rows show progress and status clearly, making the app useful during quick reviews and screen shares.

Files

Attachment previews

Images show inline previews before sending and open inside a Luvi preview modal after posting.

Settings

Identity and performance controls

Avatar customization, compact mode, presentation mode, low-motion behavior, and security controls stay in one polished panel.
